Gerenciador de requisições REST API GET POST

Este projeto fornece uma solução simples para gerenciar requisições GET e POST para APIs REST.

Visão geral do projeto

O REST API GET POST Manager é uma ferramenta projetada para simplificar o envio de requisições GET e POST para APIs REST. Oferece uma interface fácil de usar para desenvolvedores interagirem com serviços RESTful.

Recursos

  • Interface simples para fazer requisições GET
  • Formulário simples para construir requisições POST com dados JSON
  • Visualização de resposta com realce de sintaxe
  • Compatível com CORS
  • Histórico de requisições

Repositório

Encontre o código fonte e contribua no GitHub: https://github.com/albertincx/restapi-get-post-manager
