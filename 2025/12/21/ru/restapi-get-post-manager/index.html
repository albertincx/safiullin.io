Менеджер запросов REST API GET POST

Этот проект предоставляет простое решение для управления GET- и POST-запросами к REST API.

Обзор проекта

REST API GET POST Manager — это утилита, призванная упростить отправку GET- и POST-запросов к REST API. Она предлагает простой в использовании интерфейс для разработчиков, взаимодействующих с сервисами RESTful.

Возможности

  • Простой интерфейс для выполнения GET-запросов
  • Удобная форма для составления POST-запросов с данными JSON
  • Визуализация ответа с подсветкой синтаксиса
  • Совместимость с CORS
  • Отслеживание истории запросов

Репозиторий
