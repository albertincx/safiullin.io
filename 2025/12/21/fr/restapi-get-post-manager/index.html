Gestionnaire de requêtes REST API GET POST

Ce projet fournit une solution simple pour gérer les requêtes GET et POST vers les API REST.

Présentation du projet

Le REST API GET POST Manager est un outil conçu pour simplifier l’envoi de requêtes GET et POST aux API REST. Il offre une interface facile à utiliser pour les développeurs interagissant avec des services RESTful.

Fonctionnalités

  • Interface simple pour effectuer des requêtes GET
  • Formulaire facile pour construire des requêtes POST avec des données JSON
  • Visualisation de la réponse avec coloration syntaxique
  • Compatible CORS
  • Suivi de l’historique des requêtes

Dépôt

Trouvez le code source et contribuez sur GitHub : https://github.com/albertincx/restapi-get-post-manager
