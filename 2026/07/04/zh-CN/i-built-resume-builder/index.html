我仅通过一条提示词就用 AI 构建了一个简历生成器

IT Resume Builder 是一款专为 IT 专业人员量身定制的 premium 在线简历构建工具。它允许你在浏览器中直接生成、定制并导出高分辨率的精美简历。

当时我正在找工作，他们要求发送一份简历，所以我考虑构建一个新颖、现代的简历，并决定自己开发这个工具。

主要功能特点

  • 现代排版布局模板：可以选择 Modern Tech（现代技术）、Professional Sidebar Left（专业左侧栏）以及 Clean Minimalist（极简干净）等不同的模板风格。
  • 精品字体与样式：支持精致的 Google Fonts 字体组合（例如 Outfit/Inter、Space Grotesk/Inter 和 JetBrains Mono）并配有可自定义的主题颜色。
  • 完美 A4 纸张缩放：包含实时页面溢出警告，这样你可以清晰知道内容是否完美塞进标准的 A4 打印限制内。
  • 灵活的数据导出：下载 JSON 文件备份，导出为 Microsoft Word (.doc) 格式，或直接生成用于打印的干净 PDF。
  • 数据隐私与自主权：所有数据的处理与存储均完全在你的浏览器本地进行。无需注册账号，并支持随时清除或重载演示数据。
  • 双语界面：可在英文与俄文 UI 界面之间一键切换。

在这里体验：https://resume-builder.safiullin.dev/
