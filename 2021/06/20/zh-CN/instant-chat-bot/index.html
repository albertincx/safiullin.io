InstantChatBot

https://safiullin.com/projects/instant-chat-bot-for-website.html

假设你有一个网站，有人来访问。但如何让他们能快速联系到你并获得即时答复呢？

技术能帮我们解决这一问题。
配方很简单：
快速的消息传递软件 —— 用于即时回复 (Telegram)
Node —— 用于处理请求
React —— 用于创建网页端聊天窗口
免费服务器 —— 托管我们的作品。

  1. 首先，我们做一个 React 应用程序，为此我们从 GitHub 克隆一份模板并删除不必要的部分。

  2. 现在我们需要把消息发送到某处。
    Websocket 技术可以帮我们实现这一点。
    我们将把它运行在后端的 Node 服务上。

  3. 我们在 Telegram 中创建一个群组，这样所有的网站消息都会发到里面。然后我们可以直接在群里回复。

  4. 现在我们需要某种逻辑来标识网站用户。有一些简单的原则：

  • 为每个新访问者创建一个唯一的密钥 (key)
  • 将该密钥绑定到每条消息中
  • 在 Telegram 中引用该消息进行回复时，就意味着将消息发送回该网页聊天室。

一个实用的网站客服聊天窗口就做好了，用于在 Telegram 中与客户进行沟通，而且是完全免费的。
