InstantChatBot

https://safiullin.com/projects/instant-chat-bot-for-website.html

สมมติว่าคุณมีเว็บไซต์ที่มีคนเข้ามาเยี่ยมชม แต่จะทำอย่างไรให้พวกเขาติดต่อคุณได้อย่างรวดเร็วและได้รับคำตอบในทันที?

เทคโนโลยีจะช่วยเราในเรื่องนี้เอง
สูตรสร้างง่ายๆ:
แอปแชทที่รวดเร็ว - เพื่อการตอบกลับทันใจ (Telegram)
Node - สำหรับประมวลผลคำขอ
React - สำหรับสร้างหน้าต่างแชทบนเว็บ
เซิร์ฟเวอร์ฟรี - ที่ที่ผลงานของเราจะถูกนำไปติดตั้ง

  1. ขั้นแรก เราสร้างแอป React ขึ้นมา โดยนำโปรเจกต์ต้นแบบจาก GitHub มาปรับใช้และตัดส่วนที่ไม่จำเป็นออก

  2. ตอนนี้เราต้องการส่งข้อความไปที่ไหนสักแห่ง
    เทคโนโลยี WebSocket จะช่วยเราในจุดนี้
    ซึ่งเราจะรันมันบนเซิร์ฟเวอร์ด้วย Node.js

  3. สร้างกลุ่มใน Telegram เพื่อให้ข้อความทั้งหมดส่งเข้าไปที่นั่น และเราจะสามารถตอบกลับข้อความเหล่านั้นได้

  4. ตอนนี้เราต้องหาวิธีระบุตัวตนของผู้ใช้แต่ละคน มีกฎง่ายๆ ดังนี้:

  • สร้างคีย์เฉพาะ (unique key) สำหรับผู้เยี่ยมชมใหม่แต่ละคน
  • แนบคีย์นั้นไปกับทุกข้อความ
  • การตอบกลับข้อความใน Telegram จะหมายถึงการตอบกลับไปยังแชทบนเว็บ

ระบบแชทบนเว็บแสนสะดวกสำหรับสื่อสารกับลูกค้าผ่าน Telegram พร้อมใช้งานแล้ว และใช้งานได้ฟรี!
